## Markdown이란?
일반 텍스트로 서식이 있는 문서를 작성하는 방법
## Markdown 문법
---
1. Header : \#... or \##... or \###...

    # Header 1
    ## Header 2
    ### Header 3

2. Italic : \*...* or \_..._

    *Italic1*
    _Italic2_

3. Bold : \*\*...** or \_\_...__

    **Bold1**
    __Bold2__

4. Strikethrough : \~\~...~~

    ~~Strikethrough~~

5. Unordered List : \- ... or \* ...

    - Unordered List 1
    * Unordered List 2

6. Ordered List : 1. ...

    1. Ordered List 1
    2. Ordered List 2

7. Code : \'...'

    `print()`

8. Code Block : \`\`\`...```

    ```python
    def powr(x: int):
    	return x ** 2
    ```
